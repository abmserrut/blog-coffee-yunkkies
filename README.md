# Blog de Café - Coffee Yunkkies

Proyecto web desarrollado para practicar maquetación frontend, diseño responsivo y buenas prácticas de desarrollo web.

## 🚀 Tecnologías utilizadas
- HTML5
- CSS3 (con normalización)
- JavaScript (Vanilla)

## 🎯 Características
- Interfaz 100% responsiva.
- Modo oscuro/claro con persistencia en localStorage.
- Navegación dinámica.
- Estructura semántica accesible.

<a href="https://blog-coffee-yunkkies.netlify.app/" target="_blank">🚀 Ver el proyecto en vivo aquí</a>

## 🛠️ Cómo ejecutar
1. Clona este repositorio.
2. Abre `index.html` en tu navegador.
